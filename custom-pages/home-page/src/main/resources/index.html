Default application home page

This page is the default home page of any Living Application. It is a default custom page provided with Bonita BPM.

You can replace it with another custom page. For more information, see the documentation about Custom Pages.

To replace the home page, create and import a custom page as described in the documentation. Then add the custom page to your application, and click the Home Page icon to make the new page the home page. Then you can remove the default home page from your application.

When you have added a page to your application, to be able to see it, you need to reference the page in the navigation.
